# Reference material

Not part of the build. Kept for reference only.

- `enclose_original.py` — dynomight's original IP model (AGPL-3.0), the starting
  point for `enclose.py` in the repo root. Reads the map from a screenshot.
- `map.png` — the screenshot `enclose_original.py` expects.
- `play.js`, `mini-puzzle.js` — minified client bundles from enclose.horse, the
  ground truth for scoring and for how bees/portals actually behave.
